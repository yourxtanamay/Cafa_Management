# ☕ Café Management System

A simple **Café Management System** built using **Python and Tkinter GUI**.
This application allows users to select food and beverage items, calculate the total bill, add tax, and generate a receipt automatically.

The project demonstrates the use of **Python GUI programming, event handling, and basic billing logic**.

---

## 📌 Features

* Graphical User Interface using Tkinter
* Menu with different café items
* Quantity input for each item
* Automatic bill calculation
* 5% GST tax calculation
* Receipt generation with bill number and date
* Reset order option
* Exit confirmation popup

---

## 🍔 Menu Items

The system currently includes the following menu items:

* Espresso
* Cappuccino
* Latte
* Tea
* Sandwich
* Burger
* Pastry
* Muffin

Each item has a predefined price and users can enter the quantity to calculate the total bill.

---

## 🧮 How It Works

1. User enters the quantity for desired items.
2. Click **Calculate Total** to calculate the subtotal and tax.
3. Click **Generate Receipt** to display the full bill.
4. Use **Reset** to clear the order.
5. Click **Exit** to close the application.
